# GitHub Pages 產品效益 Demo

## 內含檔案
- `index.html`：完整網站，包含樣式、下拉選單與示範計算公式。

## 發布方式
1. 登入 GitHub，建立一個新的 Public repository。
2. 將 `index.html` 上傳到 repository 最上層。
3. 進入 `Settings` → `Pages`。
4. 在 `Build and deployment` 的 Source 選擇 `Deploy from a branch`。
5. Branch 選擇 `main`，資料夾選擇 `/(root)`，按 `Save`。
6. 等候 GitHub 顯示網站網址。

## 修改內容
使用文字編輯器開啟 `index.html`：
- 搜尋「下拉選單資料」修改作物、重量、箱體大小、材質與溫度。
- 搜尋「示範計算公式」修改分數、建議數量與提示邏輯。

## 重要提醒
目前公式與分數都是 Demo 示意值。正式對外使用前，應改成實際產品試驗、使用標示與法規允許的內容。
